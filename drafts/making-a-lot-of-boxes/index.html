Making a lot of boxes.

Now we have one box in our VR scene. But what if we want to make more than one box? One thing we could do is duplicate the <a-box> tag and change the position. Unfortunately, that's a lot of work if we wanted, for example, 10 boxes randomly placed throughout the scene.

To programmatically place multiple boxes at random points in the scene, we'll need to use some JavaScript. Luckily, A-Frame makes it easy to use JavaScript to place items in scenes.


    <a-scene>
        <a-sky src="/aframe/img/Rosenkranz-Basilika.jpg"></a-sky>
        <a-box position="0 1 -3" color="#23A6C7" rotation="10 45 30"></a-box>
    </a-scene>

    <script type="text/javascript">
        const scene = document.querySelector("a-scene"); // Finds the a-scene element.
        const box = document.createElement("a-box"); // Creates an a-box element.

        box.setAttribute("position", "-2 2 -3"); // Adds attributes to the box.
        scene.appendChild(box); // Adds the box to the scene.
    </script>

In this case, we're adding the JavaScript directly into the HTML page by using a <script type="text-javascript"></script> tag. We want to make sure to put it after our a-scene closing tag so we can access the a-scene and everything that has already been created inside it.
