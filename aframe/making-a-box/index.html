A-Frame

Making a box.

Once we have our background in place, we're ready to start adding some simple 3D items to our scene. Like with the sky primitive, A-Frame comes with other shapes out of the box, too. For our first shape, we'll add a simple <a-box> to the scene.


    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>A-Frame Tutorial 1</title>
        <script src="https://aframe.io/releases/0.9.0/aframe.min.js"></script>
    </head>
    <body>
        <a-scene>
            <a-sky src="/aframe/img/background.jpg"></a-sky>
            <a-box></a-box>
        </a-scene>
    </body>
    </html>

At first, it seems like nothing was added to the scene, but if you look at the floor, you'll see that the box primitive was added directly below you. The center of the box was added to coordinate "0 0 0" in the scene. Note that the camera itself isn't at "0 0 0". A-Frame sets the default camera to a Y-coordinate (or "height") of 1.6 meters, to simulate an average person's height within the scene.

We should put the box where we can easily see it in the scene. Let's put it directly in front of us (x = 0), one meter up (y = 1) and three meters away (z = -3). In A-Frame, the Z-coordinate system is negative as it extends into the screen and positive outward from the screen. If we had put the cube at z = 3, it would've been three meters behind us. You can read more about positioning in the A-Frame documentation.


    <a-scene>
        <a-sky src="/aframe/img/background.jpg"></a-sky>
        <a-box position="0 1 -3"></a-box>
    </a-scene>

It's a boring, colorless cube, but we can make it more interesting with some color and rotation. Rotation is set by degrees, and you can read more about it in the documentation.


    <a-scene>
        <a-sky src="/aframe/img/background.jpg"></a-sky>
        <a-box position="0 1 -3" color="#23A6C7" rotation="10 45 30"></a-box>
    </a-scene>

Now we have a much more interesting scene that doesn't look as flat and static as it was before. Next, we'll add some more cubes with different attributes to our scene.

Documentation

  • A-Frame position: https://aframe.io/docs/0.9.0/components/position.html
  • A-Frame rotation: https://aframe.io/docs/0.9.0/components/rotation.html
